### 3. 按圆锥曲线类型分类的定理

#### **🔵 椭圆专属定理（共15条）**

1. **椭圆第一定义**：|PF₁| + |PF₂| = 2a
2. **椭圆第二定义**：|PF|/d = e（d为到准线距离）
3. **椭圆标准方程**：
   - 焦点在x轴：x²/a² + y²/b² = 1
   - 焦点在y轴：y²/a² + x²/b² = 1
4. **椭圆参数关系**：a² = b² + c²
5. **椭圆离心率**：e = c/a，其中 0 < e < 1
6. **椭圆焦半径**：
   - |PF₁| = a + ex₀
   - |PF₂| = a - ex₀
7. **椭圆通径**：2b²/a
8. **椭圆焦点三角形面积**：S = b²·tan(∠F₁PF₂/2)
9. **椭圆准线方程**：x = ±a²/c
10. **椭圆上点到焦点距离范围**：[a-c, a+c]
11. **椭圆焦点三角形周长**：2a + 2c
12. **椭圆参数方程**：x = acosθ, y = bsinθ
13. **椭圆切线方程**：x₀x/a² + y₀y/b² = 1
14. **椭圆中点弦斜率关系**：k·k_{OM} = -b²/a²
15. **椭圆焦点弦最值**：最短弦为通径，最长弦为长轴

#### 🔴 双曲线专属定理（共15条）

1. **双曲线第一定义**：||PF₁| - |PF₂|| = 2a
2. **双曲线第二定义**：|PF|/d = e（d为到准线距离）
3. **双曲线标准方程**：
   - 焦点在x轴：x²/a² - y²/b² = 1
   - 焦点在y轴：y²/a² - x²/b² = 1
4. **双曲线参数关系**：c² = a² + b²
5. **双曲线离心率**：e = c/a，其中 e > 1
6. **双曲线渐近线方程**：y = ±(b/a)x
7. **双曲线焦点到渐近线距离**：d = b
8. **双曲线通径**：2b²/a
9. **双曲线焦点三角形面积**：S = b²/tan(∠F₁PF₂/2) 或 S = b²·cot(θ/2)
10. **双曲线准线方程**：x = ±a²/c
11. **双曲线右支上点横坐标范围**：x ≥ a
12. **双曲线共渐近线系**：x²/a² - y²/b² = λ
13. **双曲线顶点到渐近线距离**：d = ab/c
14. **双曲线焦半径最小值**：c - a（右支到右焦点）
15. **等轴双曲线**：a = b时，e = √2

#### 🟢 抛物线专属定理（共12条）

1. **抛物线定义**：|PF| = d（到焦点距离=到准线距离）
2. **抛物线标准方程**：
   - y² = 2px（开口向右）
   - y² = -2px（开口向左）
   - x² = 2py（开口向上）
   - x² = -2py（开口向下）
3. **抛物线焦点坐标**：F(p/2, 0) 或 F(0, p/2)
4. **抛物线准线方程**：x = -p/2 或 y = -p/2
5. **抛物线焦半径公式**：|PF| = x₀ + p/2（标准形式）
6. **抛物线焦点弦长**：|AB| = x₁ + x₂ + p = p/sin²θ
7. **抛物线通径**：2p（最短焦点弦）
8. **抛物线焦参数**：p = 焦点到准线距离
9. **抛物线焦点弦端点横坐标乘积**：x₁x₂ = p²/4
10. **抛物线焦点弦端点纵坐标乘积**：y₁y₂ = -p²
11. **抛物线切线方程**：y₀y = 2p(x + x₀)
12. **抛物线导数**：y' = x/p

#### ⚪ 通用定理（所有曲线适用）

1. **余弦定理**：c² = a² + b² - 2ab·cosC
2. **正弦定理**：a/sinA = b/sinB = c/sinC
3. **弦长公式**：|AB| = √(1+k²)·√[(x₁+x₂)²-4x₁x₂]
4. **点到直线距离**：d = |Ax₀+By₀+C|/√(A²+B²)
5. **中点坐标公式**：((x₁+x₂)/2, (y₁+y₂)/2)
6. **斜率公式**：k = (y₂-y₁)/(x₂-x₁)
7. **两点距离公式**：d = √[(x₂-x₁)² + (y₂-y₁)²]
8. **三角形面积公式**：
   - S = (1/2)·底·高
   - S = (1/2)ab·sinC
   - S = (1/2)|x₁y₂ - x₂y₁|
9. **向量数量积**：
   - 代数形式：a⃗·b⃗ = x₁x₂ + y₁y₂
   - 几何形式：a⃗·b⃗ = |a⃗||b⃗|cosθ
10. **垂直关系**：k₁·k₂ = -1 或 a⃗·b⃗ = 0
11. **判别式Δ**：Δ = b² - 4ac
12. **三角形中位线**：中位线 = 底边/2
13. **等积法**：同底（或同高）面积相等
14. **内切圆半径**：r = 2S/周长

### 4. 高频技巧

#### **特殊技巧：**

1. **设直线方程为 x = my + n 形式**（频率极高）
   - 避免斜率不存在的讨论
   - 特别适合抛物线问题

2. **利用"当且仅当"求最值**（频率很高）
   - 基本不等式取等条件
   - 几何最值问题

3. **焦点弦的性质**（频率很高）
   - 焦点弦长公式：|AB| = 2p/sin²θ
   - x₁x₂ = p²/4, y₁y₂ = -p²

4. **共渐近线双曲线系**（频率高）
   - 形如 x²/a² - y²/b² = λ

5. **齐次化求离心率**（频率高）
   - 将a,b,c关系式两边同除以a²

6. **向量法处理垂直关系**（频率高）
   - a⃗·b⃗ = 0
