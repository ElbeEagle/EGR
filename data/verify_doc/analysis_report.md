# 圆锥曲线题目定理序列对应性分析报告

**日期**: 2026-01-12
**分析范围**: 全量数据集 (5357个含过程的题目)

## 1. 概述

本报告旨在分析题目文本 (`Text`)、解题过程 (`Process`) 与定理模型序列 (`Model Sequence`) 之间的对应关系。通过对全量数据的启发式扫描，我们发现大部分数据标注准确，但存在一定比例的遗漏和不一致现象。

## 2. 统计摘要

- **总分析题目数**: 5357
- **存在潜在对应问题的题目数**: 680 (约 12.7%)
- **主要问题类型分布**:
  - **切线/判别式模型缺失**: ~193 例 (文本含"切线"或"相切"，但无Tangent/Discriminant模型)
  - **离心率模型缺失**: ~179 例 (文本含"离心率"，但无Eccentricity模型)
  - **双曲线标签缺失**: ~152 例 (文本含"双曲线"，但无Hyperbola模型)
  - **渐近线模型缺失**: ~97 例 (文本含"渐近线"，但无Asymptote模型)
  - **椭圆标签缺失**: ~86 例 (文本含"椭圆"，但无Ellipse模型)
  - **抛物线标签缺失**: ~77 例

> *注：一个题目可能包含多个标签缺失问题。*

## 3. 典型问题分析

### 3.1 严重缺失 (模型列表为空或极少)

部分题目虽然过程完整，但模型序列为空或仅含有一个无关模型。

* **Index 1540**:

  * **题目**: 已知双曲线的离心率为$\sqrt{2}$，且与椭圆...相同焦点...
  * **模型序列**: `[]` (空)
  * **问题**: 严重缺失。应包含 `Hyperbola_Standard_Equation`, `Ellipse_Standard_Equation`, `Eccentricity` 等。
* **Index 4190**:

  * **题目**: 双曲线焦点到渐近线距离...
  * **模型序列**: `[]` (空)
  * **问题**: 严重缺失。

### 3.2 曲线类型标签错误 (错标)

题目讨论的是某一种曲线，但标签却全是另一种曲线的模型。

* **Index 201**:
  * **题目**: 与双曲线...有相同焦点...的椭圆方程...
  * **模型序列**: `Parabola_Equation_Standard_Right`, `Parabola_Definition` (抛物线模型)
  * **问题**: 题目涉及双曲线和椭圆，但标签全是抛物线。这可能是数据增强或ID映射时的错误。

### 3.3 关键几何条件缺失

文本中明确提到的几何条件（如切线、渐近线）在模型中未体现。

* **Index 768**:
  * **题目**: 双曲线...渐近线与圆...相切...
  * **模型序列**: `Hyperbola_Equation_Standard_X`, `Hyperbola_Asymptote`, `Point_To_Line_Distance`
  * **问题**: 缺少切线相关模型（尽管用了点到直线距离来处理切线，但标准做法通常会标记切线或判别式模型以体现语义）。

### 3.4 隐式概念未标记

* **Index 2695**:
  * **题目**: 双曲线离心率互为倒数...
  * **模型序列**: `Basic_Inequality`
  * **问题**: 缺少 `Hyperbola` 和 `Eccentricity` 的基础标签，仅标注了求解过程中用到的不等式。

## 4. 结论与建议

1. **数据清洗**: 建议重点审查上述 flagged (680个) 的样本，特别是模型列表为空 (`[]`) 的样本。
2. **标签补全**: 对于包含明确关键词（如"离心率", "渐近线"）的题目，应强制检查是否包含对应的 ID。
3. **交叉验证**: 注意检查同时包含多种曲线名称的题目，确保所有涉及的曲线模型都被召回，避免出现"题文是椭圆，标签是抛物线"的错误。

附件: 详细的采样问题请见 `sampled_problems.md`。
